# CutLang ATLAS Open Data Analysis

## Overview

This repository contains analysis work using CutLang on ATLAS 13 TeV Open Data. It includes BSM physics studies, custom analysis definitions, documentation, and visualization tools for exploring particle physics datasets.


## Analyses

For CutLang syntax and workflow documentation, see the [CutLang documentation](https://github.com/unelg/CutLang).

## Directory Structure

- `13TevATLASOpenDatarelease/` – ROOT files, analysis definitions (`.adl`), plotting scripts, and LaTeX reports
- `SUSY/` – SUSY-specific analysis studies
